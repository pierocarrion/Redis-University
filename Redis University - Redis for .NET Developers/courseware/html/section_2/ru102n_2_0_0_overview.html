Unit 2 Overview

Welcome to Unit 2 of RU102N: Redis for .NET Developers. In the previous unit we covered the basics of how to connect and use the StackExchange.Redis library. In this unit we will get into the nitty gritty of how to use various Redis Data structures and features from StackExchange.Redis. In this unit we will cover:

  • Strings
  • Lists
  • Sets
  • Sorted Sets
  • Hashes
  • Streams
  • Scripts
  • Transactions
  • Pub/Sub

We've linked to each of the relevant articles for each concept on redis.io. If you want a full explanation of each of these, and you haven't taken RU101: Introduction to Redis Data Structures, that course has full explanations for each of these concepts, and can be a great resource.

In this unit we will briefly introduce the concepts and demonstrate the mechanics of using them using the StackExchange.Redis library.

Hello and welcome to Redis University. This is RU102N, Redis for .NET Developers.

First let me introduce myself. I'm Steve Lorello, I'm a Senior Field Engineer at Redis and I'm here to help guide you along the way as you dive into using Redis from .NET. Before we begin, I want to provide a rough overview of what we'll be covering in this class. This class is all about the interaction of two technologies .NET and Redis. In this course we'll learn:

  • About the primary client of the .NET ecosystem StackExchange.Redis
  • How to interact with all the data structures you learned about in RU101: Introduction to Redis Data Structures in .NET
  • How Redis melds beautifully into some of the core components of ASP.NET and ASP.NET core to handle key use cases such as Caching and Session State Management
  • How you can interact with the various extended data structures offered by Redis Stack in .NET
  • How to model, index, and query items in Redis using LINQ and Redis OM .NET

Throughout the course I'll have a lot of hands on exercises for you to work through. These sections are not graded, but I highly recommend you invest time in doing each of them. After all, there's no better way to learn than to do.
